Detailed Description

Clase base abstracta para todas las curvas de tasas de interés.

Author
Alvaro Díaz

Esta clase define varios métodos que las clases derivadas deben implementar. Los distintos tipos de curvas (descuento, proyección, por factores o tasas) se derivan de ésta.

Enumerator
qcProjectingCurve 

Curva de proyección. Al interpolar se obtiene una tasa forward de un tenor predefinido.

Curva cero cupón. Se especifica entregando los plazos en días y valores de las tasas.

qcDiscountFactorCurve 

Curva de factores de descuento. Se especifica entregando los plazos en días y los valores de los factores de descuento.
